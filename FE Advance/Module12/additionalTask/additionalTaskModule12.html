Модуль 12 - Дополнительное задания

Писать сам gulpfile.js итд не надо, просто в codepen наберите код, который будет синтаксически верным и рабочим, если использовать его в конфигурации Gulp.

  • Написать простую Gulp задачу, с именем 'TS', которая берет все файлы с раширением .ts из папки A и выбрасывает в папку B.
  • Написать простую Gulp задачу слежения, с именем 'watch:ts', которая следит за всеми файлами в папке A, и при изменении любого файла с раширением .ts, выполняет таск с именем 'TS'.
  • Написать npm скрипт с именем 'build:ts', который выполняет gulp задачу с имнем 'watch:ts'
